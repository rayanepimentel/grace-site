Ciência de Dados para Ensino Médio

Introduzindo estudantes ao mundo dos dados, análise e visualização de informações

Descobrindo o Poder dos Dados

O projeto Ciência de Dados para Ensino Médio tem como objetivo introduzir estudantes ao universo da análise de dados, estatística aplicada e visualização de informações, preparando-os para os desafios do mundo digital e profissional.

  • Introdução aos conceitos fundamentais de Ciência de Dados
  • Análise exploratória de dados com ferramentas práticas
  • Visualização de dados e criação de gráficos informativos
  • Estatística básica aplicada a problemas reais
  • Introdução à programação em Python para análise de dados
  • Projetos práticos com datasets reais

As oficinas são ministradas por estudantes e pesquisadores do ICMC-USP com experiência em Ciência de Dados, utilizando ferramentas modernas e acessíveis como Python, Jupyter Notebooks e bibliotecas populares de análise de dados.

Conteúdo Programático

O curso aborda desde conceitos básicos até aplicações práticas, incluindo:

  • Coleta e organização de dados
  • Limpeza e preparação de datasets
  • Análise estatística descritiva
  • Criação de visualizações impactantes
  • Interpretação de resultados
  • Apresentação de insights

Para Quem é Este Curso

O curso é voltado para estudantes do ensino médio que têm curiosidade sobre dados, estatística e tecnologia. Não é necessário conhecimento prévio em programação, pois começamos do básico e avançamos gradualmente.

Como Participar

As turmas são abertas periodicamente e divulgadas em nossas redes sociais. Para mais informações ou para levar este projeto para sua escola, entre em contato através do e-mail grace@icmc.usp.br.
