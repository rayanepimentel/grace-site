Programação Web para Meninas e Mulheres

A Escola de Web para Meninas é um dos projetos mais importantes do GRACE, oferecendo cursos completos de desenvolvimento web para meninas e mulheres que desejam ingressar no mundo da programação e criar seus próprios sites e aplicações.

  • Fundamentos de HTML5 - Estrutura de páginas web
  • CSS3 - Estilização e design responsivo
  • JavaScript - Interatividade e programação
  • Frameworks modernos e ferramentas de desenvolvimento
  • Projetos práticos e portfólio pessoal
  • Publicação de sites na internet

O curso é ministrado por desenvolvedoras e estudantes do ICMC-USP com experiência em desenvolvimento web, proporcionando um ambiente acolhedor e inspirador para o aprendizado.

Metodologia

Utilizamos uma abordagem prática e hands-on, onde as participantes aprendem fazendo. Cada conceito é seguido de exercícios práticos e projetos reais que vão compor o portfólio das alunas.

O que você vai criar

  • Página pessoal/portfólio
  • Landing pages responsivas
  • Formulários interativos
  • Jogos simples em JavaScript
  • Aplicações web dinâmicas

Público-Alvo

O curso é voltado para meninas e mulheres a partir de 14 anos, sem necessidade de conhecimento prévio em programação. Oferecemos turmas para diferentes níveis: iniciante, intermediário e avançado.

Inscrições

As inscrições são abertas periodicamente e divulgadas em nossas redes sociais. Fique atenta ao Instagram @grace.icmc.usp para não perder as próximas turmas!
