# Assets Necessários

Este diretório contém as imagens do site GRACE ICMC-USP. Os seguintes arquivos ainda precisam ser adicionados:

## Arquivos Faltantes

### 1. favicon.png
- **Localização:** `assets/img/favicon.png`
- **Descrição:** Ícone que aparece na aba do navegador
- **Dimensões recomendadas:** 32x32 pixels ou 16x16 pixels
- **Formato:** PNG com fundo transparente
- **Conteúdo sugerido:** Logo do GRACE ou iniciais "G"

### 2. apple-touch-icon.png
- **Localização:** `assets/img/apple-touch-icon.png`
- **Descrição:** Ícone para dispositivos Apple (iPhone, iPad)
- **Dimensões recomendadas:** 180x180 pixels
- **Formato:** PNG
- **Conteúdo sugerido:** Logo do GRACE em alta resolução

### 3. logo.png
- **Localização:** `assets/img/logo.png`
- **Descrição:** Logo principal do GRACE que aparece no cabeçalho do site
- **Dimensões recomendadas:** Largura máxima de 200px, altura proporcional
- **Formato:** PNG com fundo transparente
- **Conteúdo:** Logo oficial do GRACE ICMC-USP

## Como Adicionar os Arquivos

1. Obtenha os arquivos de logo/identidade visual do GRACE
2. Redimensione conforme as especificações acima
3. Salve os arquivos com os nomes exatos mencionados
4. Coloque-os no diretório `assets/img/`

## Estrutura Atual de Diretórios

```
assets/img/
├── blog/              # Imagens para posts do blog
├── colaboradores/     # Fotos dos colaboradores do GRACE
├── instagram/         # Posts do Instagram
├── masonry-portfolio/ # Imagens do portfólio
├── midia/            # Imagens de mídia/imprensa
├── pesquisa/         # Imagens de pesquisas
├── portfolio/        # Portfolio items
├── team/             # Fotos da equipe
└── testimonials/     # Fotos de depoimentos
```

## Observações

- Todos os arquivos de imagem devem ser otimizados para web
- Use compressão PNG para manter a qualidade com tamanho reduzido
- Mantenha a identidade visual do GRACE ICMC-USP em todos os assets
